# Phase 1 model weights

Ultralytics auto-downloads YOLOv8n to this directory on first run (~6MB).

## Where to get weights
- YOLOv8n (detector): https://github.com/ultralytics/assets/releases/download/v8.4.0/yolov8n.pt
- Later phases (download on demand):
  - YOLOv8n-pose: https://github.com/ultralytics/assets/releases/download/v8.4.0/yolov8n-pose.pt
  - OSNet-x0.25 (ReID): https://kaiyangzhou.github.io/deep-person-reid/MODEL_ZOO
  - SCRFD: https://github.com/deepinsight/insightface/releases
  - MobileFaceNet: https://github.com/deepinsight/insightface/releases

Weights are gitignored. Do not commit them.
